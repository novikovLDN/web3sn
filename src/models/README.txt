Положите сюда свою 3D-модель: model.glb (или .gltf).
Она автоматически появится в рендере на экране услуги «3D-моделинг»
(справа, вместо персонажа-заглушки).

• Формат: .glb (рекомендуется) или .gltf. Экспорт из Blender: File → Export → glTF 2.0 (.glb).
• Если в модели есть анимация — она проиграется автоматически (в цикле).
  Если анимации нет — модель будет медленно вращаться на «поворотном столе».
• Размер файла: желательно до ~8–10 МБ (оптимизируйте/сожмите, напр. Draco).
• Модель автоматически центрируется и масштабируется под сцену — размер не важен.
• Файлов может быть несколько — возьмётся первый по алфавиту (нумеруйте: 01-...).
