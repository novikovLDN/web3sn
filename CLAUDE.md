# Портфолио NOVIKOV — контекст проекта

**Перед первой правкой прочитай `ТЗ.md` в корне.** Там позиционирование,
формула экранов, тон голоса, накопленные грабли и что уже сделано. Здесь —
только то, что обязано быть в контексте с первой секунды.

## Что это

Сайт **senior/lead дизайнера-разработчика**. Задача — сообщить уровень и цену,
а не «показать работы». Ключевые слова заказчика: дорого, недоступно,
технологично, погружение.

## Окружение

```bash
export PATH="$HOME/.local/node/bin:$PATH"   # Node не в системном PATH
./node_modules/.bin/tsc --noEmit && ./node_modules/.bin/vite build
```

Сборка = `tsc && vite build`. **Неиспользуемая переменная валит деплой.**
Playwright намеренно не в `package.json` — живёт только локально.

## Git

Работаем на `main`. **Коммитить и пушить сразу** после проверенной порции
изменений, не спрашивая. Сверять ветку: `git push origin main` с фича-ветки
пушит локальный `main`, а не текущую работу, и делает это молча.

## Пять правил, которые нельзя нарушать

1. **Только токены.** Ни одного сырого хекса, произвольного отступа или
   длительности в компонентах. Цвет — `var(--a)`, отступ — `var(--s-8)`,
   кривая — из `src/design/motion.ts`.
2. **Анимировать только `transform` и `opacity`.** Всё остальное вызывает
   layout или paint. Заливки — через `scaleX/scaleY` на отдельном слое.
3. **Ничего не выдумывать про человека.** Ни клиентов, ни наград, ни кейсов,
   ни цифр результата, ни отзывов, ни дат. Пустые данные скрываются кодом,
   а не заполняются правдоподобным вымыслом.
4. **Проверять глазами, а не отчётами.** Скриншоты на 1400 и 390 px ловят то,
   что проходит тайпчек: схлопнутый пробел, срезанную «й», наезжающий
   заголовок, слово, разорванное посреди букв.
5. **Дорогой сайт может быть насыщенным, но не громким.** «Больше элементов»
   делается как плотность реакции, а не плотность объектов.

## Грабли, на которые уже наступали

- **Кириллица:** интерлиньяж ниже `0.88` срезает «й» и «ё». Премиальные
  гарнитуры часто не имеют кириллицы вообще — проверять по метаданным
  Google Fonts (пойманы Instrument Serif, Bricolage Grotesque).
- **`overflow-hidden` на предке ломает `position: sticky`.**
- **`transform: scale` не уменьшает бокс в потоке** — даёт горизонтальное
  переполнение.
- **Ширина анимируемого marquee — под ~12000px**, дальше слой пропадает.
- **JSX-комментарий внутри открывающего тега** — синтаксическая ошибка.
- **`rAF` и `getBoundingClientRect` в кадре** — измеренная причина падения
  fps. Гейтить по `IntersectionObserver`, геометрию кэшировать.
- **Условный вызов хука** (`inFrame && useHook()`) ломает React.
